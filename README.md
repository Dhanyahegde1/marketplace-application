# Marketplace Flask Application

A role-based marketplace web application built using Flask.

## Features
- User Authentication (Register / Login)
- JWT-based Authorization
- Buyer Module:
  - View resources
  - Purchase resources
  - View transaction history
- Seller Module:
  - Add resources
  - Delete resources
- Admin Module:
  - Manage users
  - Manage resources
  - View all transactions

## Tech Stack
- Python
- Flask
- Flask-JWT-Extended
- SQLAlchemy
- SQLite
- HTML, CSS, JavaScript

## How to Run
```bash
pip install -r requirements.txt
python main.py
